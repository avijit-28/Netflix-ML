# Task 2 (Easy) – Content Type Prediction Model

## Description
Develop a machine learning model to predict whether a Netflix title is a Movie or TV Show based on its available features.

## Workflow
1. Selected relevant dataset features (director, country, rating, genres, release year)
2. Encoded categorical variables using LabelEncoder
3. Trained Logistic Regression and Random Forest classifiers
4. Evaluated prediction performance (train vs test accuracy, checked for overfitting)
5. Compared model accuracy in a summary table

## Skills Demonstrated
Classification Algorithms, Data Encoding, Model Validation, Machine Learning Workflow

## How to Run
Open the notebook in Google Colab, upload `Dataset.csv`, and run all cells in order.
