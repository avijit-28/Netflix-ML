# Task 5 (Advanced) – Netflix Trend Forecasting Model

## Description
Develop a machine learning model to forecast future Netflix content release trends based on historical data.

## Workflow
1. Prepared time-based features from `date_added`
2. Analyzed historical monthly content addition patterns
3. Built forecasting models: Linear Regression + Moving Average baseline
4. Generated 6-month future predictions with visualization
5. Evaluated forecasting accuracy using MAE and RMSE

## Skills Demonstrated
Time Series Forecasting, Predictive Analytics, Model Evaluation, Trend Analysis

## How to Run
Open the notebook in Google Colab, upload `Dataset.csv`, and run all cells in order.
