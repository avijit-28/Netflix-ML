# Task 4 (Medium) – Netflix Content Segmentation

## Description
Group Netflix titles into meaningful clusters using unsupervised machine learning techniques.

## Workflow
1. Prepared numerical and categorical features
2. Scaled features and applied K-Means clustering (k=5)
3. Identified and profiled content groups per cluster
4. Visualized clusters in 2D using PCA
5. Interpreted and summarized each cluster's defining characteristics

## Skills Demonstrated
K-Means Clustering, Data Scaling, Cluster Analysis, Unsupervised Machine Learning

## How to Run
Open the notebook in Google Colab, upload `Dataset.csv`, and run all cells in order.
