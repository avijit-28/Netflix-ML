# Task 3 (Medium) – Netflix Audience Rating Classification

## Description
Build a model that predicts the audience rating category of Netflix content using content attributes.

## Workflow
1. Analyzed rating category distribution (14 unique categories, imbalanced)
2. Prepared and encoded training features/target
3. Trained Decision Tree and Random Forest classifiers
4. Optimized Random Forest via GridSearchCV hyperparameter tuning
5. Evaluated prediction accuracy with a full classification report

## Results
- Decision Tree: 44.0% test accuracy
- Random Forest: 50.6% test accuracy
- Tuned Random Forest: 54.4% test accuracy

## Skills Demonstrated
Decision Trees, Random Forest, Classification Metrics, Hyperparameter Tuning

## How to Run
Open the notebook in Google Colab, upload `Dataset.csv`, and run all cells in order.
