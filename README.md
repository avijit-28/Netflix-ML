# Netflix Machine Learning Internship Projects
**Auspify Technologies — Machine Learning Internship Program**

This repository contains all 6 completed Machine Learning tasks from the Auspify Technologies internship, built using Python and scikit-learn on a Netflix titles dataset.

## About the Internship
A 4-week practical ML internship focused on predictive modeling, data analysis, algorithm development, and problem-solving skills. Completion requires any 4 of 6 tasks — all 6 were completed here.

## Dataset
`data/Dataset.csv` — Netflix titles dataset (8,790 rows) with columns: `show_id, type, title, director, country, date_added, release_year, rating, duration, listed_in`.

## Tasks Completed

| # | Task | Difficulty | Type | Key Techniques |
|---|------|-----------|------|-----------------|
| 1 | [Netflix Content Recommendation System](./Task1_Recommendation_System) | Easy | Content-Based Filtering | TF-IDF, Cosine Similarity |
| 2 | [Content Type Prediction Model](./Task2_Content_Type_Prediction) | Easy | Binary Classification | Logistic Regression, Random Forest |
| 3 | [Netflix Audience Rating Classification](./Task3_Rating_Classification) | Medium | Multi-Class Classification | Decision Tree, Random Forest, GridSearchCV |
| 4 | [Netflix Content Segmentation](./Task4_Content_Segmentation) | Medium | Clustering | K-Means, PCA, StandardScaler |
| 5 | [Netflix Trend Forecasting Model](./Task5_Trend_Forecasting) | Advanced | Time Series Forecasting | Linear Regression, Moving Average |
| 6 | [Netflix Content Success Analytics Engine](./Task6_Success_Analytics_Engine) | Advanced | End-to-End ML Pipeline | Multi-model comparison, Feature Importance, Automated Insights |
