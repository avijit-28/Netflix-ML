# Task 1 (Easy) – Netflix Content Recommendation System

## Description
Build a machine learning-based recommendation system that suggests similar Netflix titles based on genres, categories, and content attributes.

## Workflow
1. Prepared content-related features (combined genres, director, country, type, rating into a text "soup")
2. Converted text data into machine-readable format using TF-IDF vectorization
3. Calculated content similarity scores using cosine similarity
4. Generated top-5 recommendations for selected titles
5. Evaluated recommendation quality via genre overlap scoring

## Skills Demonstrated
Recommendation Systems, NLP Fundamentals, Feature Engineering, Scikit-Learn

## How to Run
Open `Task1_Netflix_Recommender.ipynb` in Google Colab, upload `Dataset.csv` when prompted, and run all cells in order.
